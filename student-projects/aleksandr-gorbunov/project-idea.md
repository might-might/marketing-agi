# Simple Presence Studio / Ku — Project Idea

## Project Overview

**Simple Presence Studio (SPS)** — bootstrapped IP-студия, создающая медитативные игры и контент вокруг практики осознанного присутствия. Философия: Wu Wei (недеяние), дзен, созерцание. Флагманский продукт — браузерная игра **Ku** (серия "Not-Hunting").

## Product: Ku

**Ku** — медитативная игра-головоломка. Игрок ищет "негативное пространство" — пустоту между объектами — и совмещает контуры с формами вокруг. Это инверсия figure-ground: вместо поиска объектов (как в hidden object) — поиск пустоты.

### Ключевые механики

- **Negative Space Discovery** — распознавание формы пустоты между объектами (figure-ground inversion)
- **Contour Matching** — выбор абстрактного контура из панели и совмещение с пустотой на изображении
- **Contemplative Flow** — безвременной геймплей без таймеров и штрафов

### Core Game Loop (7 шагов)

1. LoadLevel — загрузка изображения + масок пустот + контуров
2. DisplayScene — показ арта с очерченными пустотами
3. AwaitContourSelection — выбор контура из нижней панели
4. AwaitKuClick — нахождение совпадающей пустоты на изображении
5. ValidateMatch — проверка совпадения
6. ProvideFeedback — звук коши + анимация заполнения светом
7. CheckCompletion — все пустоты найдены → цитата → следующий уровень

### Мета-прогрессия

- **Ближний план (эволюция клетки):** простая клетка → органеллы → специализация → ткань → орган → организм
- **Дальний план (терраформинг планеты):** голая порода → атмосфера → вода → микробы → сложная жизнь

### Платформы и технологии

- **Демо:** https://might-might.ru/demos/
- **Web-first** (не mobile-first): Telegram Mini App, PWA, Yandex.Games, VK Mini Apps, itch.io, собственный домен
- **6-12 мес:** нативные iOS/Android (Capacitor/React Native)
- **Стек MVP:** Phaser 3 (~150-300KB) + HTML5
- **Контент-пайплайн:** AI-генерация (Midjourney для визуала, Suno для ambient-звуков)
- **Целевой объём:** 30 уровней на запуске, 100+ к осени

## Target Audience

### B2C (основная)
- Люди 25-45 лет, городские, стресс/выгорание от работы
- Пробовали Calm/Headspace → бросили (audio fatigue — устали от голосовых медитаций)
- Ищут медитацию, которая вовлекает, а не скучные аудиозаписи
- Любители медитативных / ambient-игр (Monument Valley, Prune, Alto's Odyssey, Playne, Loona)

### B2B (вторичная)
- HR-отделы компаний (wellness-программы)
- Боль: 70-80% пользователей Calm Business уходят за <1 месяц
- Нужны доказательства эффективности + метрики

## Business Model

- **Freemium:** 5-10 бесплатных уровней + подписка ($4.99/мес, ₽399; $49.99/год, ₽3999)
- **B2B:** $3-5/сотрудник/мес (корпоративные wellness-контракты)
- **Дистрибуция MVP:** Telegram (0% CAC, 950M аудитория), затем ASO + перфоманс-маркетинг
- Целевая выручка: 200 млн руб/год (долгосрочно, ~46.5K платящих подписчиков)

## Core Problem (Customer Perspective)

Люди хотят расслабиться и снять стресс, но:
- **Медитационные приложения** (Calm, Headspace) теряют пользователей — audio fatigue, D30 retention 3-8%
- **"Расслабляющие" игры** всё равно создают стресс через таймеры, рейтинги, соревновательные механики
- **Пустая ниша:** нет ни одной мобильной meditation-game с интерактивной механикой + подпиской + встроенным замером эффективности

## Научная база

- Снижение DMN (сети пассивного режима) при наблюдении пустот (Brewer 2011)
- Рост альфа-волн — целевое состояние расслабления (Tang 2015)
- Figure-ground inversion = когнитивный сдвиг, аналогичный медитации
- 5-15 мин сессии эффективны (Nature 2024)
- **Целевые метрики:** снижение тревожности на 15%+ (шкала STAI), рост альфа-волн +20%
