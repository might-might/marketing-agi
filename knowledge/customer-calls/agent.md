# Customer Call Processing Agent

## Когда применять агента

Применять когда пользователю нужно обработать файлы с записями звонков с клиентом.

## Agent Jobs

### Транскрибировать аудиофайл 

**Описание**: Данный Job переводит один или нескольо аудиофайлов в текстовый формат.

**Ситуация**: Есть один или несколько аудиофайлов в формате mp3, wav, ogg, которые нужно преобразовать в текстовый файл с помощью Speech-To-Text.

**Примеры запроса**: транскрибируй файл, транскрибируй все файлы в папке, транскрибируй новые файлы в папке, запусти speech to text, transcribe

**Действия**: выполнить `workflows/voice-transcription-rulebook.md`

**Ожидаемый результат**: файл(ы) в md формате в той же папке что и исходный (если не указано куда сохранять), либо в той папке которую указали для сохранения.

---

### Обработать транскрипцию звонка с клиентом

**Описание**: Данный Job анализирует текстовую транскрипцию звонка с клиентом, суммаризует ее и выделяет важные цитаты клиента (обнаруженная потребность или возражение).

**Ситуация**: Пользователь передает один или несколько аудиофайлов в формате mp3, wav, ogg, и просит преобразовать его в текстовый файл с помощью Speech-To-Text.

**Примеры запроса**: обработай транскрипцию звонка, вытащи потребности из диалога, обнови базу звонков используя файл транскрипции.

**Действия**: выполнить `workflows/customer-call-rulebook.md`

**Ожидаемый результат**: обновленные файлы с суммаризацией звонков и обновленная общая база звонков с пользователями.
