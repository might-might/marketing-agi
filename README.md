# Marketing AGI

Проект на базе Knowlifier Cognitive Components для маркетинговых задач с поддержкой AI-агентов в Agentic IDE (Cursor, Cline, Windsurf и др.).

## Возможности

- **Транскрипция звонков** — преобразование аудио (mp3, wav, ogg) в текст с помощью Speech-To-Text
- **Анализ звонков с клиентами** — суммаризация диалогов, выделение потребностей и возражений клиентов
- **База звонков** — обновление и ведение общей базы по результатам обработки
- **Создание лендингов** — генерация и обновление landing page на основе валидированных потребностей из call-database или по инструкциям пользователя

## Структура проекта

```
marketing-agi/
├── knowledge/              # Cognitive Components
│   ├── customer-calls/     # Обработка звонков с клиентами
│   ├── landing-pages/     # Лендинги
│   └── _sandbox/          # Экспериментальные компоненты (design-time)
├── student-projects/      # Рабочие проекты студентов
└── IKNOW.md          # Конфигурация компонентов и прав доступа
```

## Быстрый старт

Проект настроен для работы в Cursor. Агенты подключаются через правила в `.cursor/rules/` и используют знания из `knowledge/`.

### Примеры запросов в чате

- *«Транскрибируй файл»* / *«Транскрибируй все файлы в папке»* — преобразование аудио в текст
- *«Обработай транскрипцию звонка»* / *«Вытащи потребности из диалога»* — анализ и суммаризация звонка
- *«Обнови базу звонков используя файл транскрипции»* — обновление базы по результатам обработки
- *«Создай лендинг»* / *«Сгенерируй landing page»* / *«Обнови лендинг на основе потребностей»* — создание или обновление landing.html в проекте студента
